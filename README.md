# A Repository of the Papers Address Vehicle Routing Problems Using Neural Combinatorial Optimization Solvers
This repository provides an up-to-date list of studies addressing Vehicle Routing Problems (VRPs) using Neural Combinatorial Optimization (NCO) solvers. It follows the taxonomy provided in our paper "[Neural Combinatorial Optimization Solvers for Vehicle Routing Problems: A Survey]()". Please cite this paper if you benefit from this repository.

```
@misc{wu_2024,
title = {Neural Combinatorial Optimization Solvers for Vehicle Routing Problems: A Survey},
note={arXiv},
author = {Wu, Xuan and Wang, Di and Xiao, Yubin and Wu, Yuesong and Yu, Chaoyu and Zhou You},
year = {2024}
}
```
## How To Request the Addition of A Paper
If you know of a paper that addresses an imbalance problem concerning generic object detection and is not on this repository, you are welcome to request the addition of that paper by submitting a pull request.
